This repository showcases a **Sales Performance Dashboard** created in Excel. The dashboard visualizes sales data by product, region, and date, along with interactive charts and a slicer for filtering by region.

## Features
- **Sales by Product:** Column chart showing sales for each product.
- **Sales by Region:** Line chart to track regional sales performance.
- **Daily Sales Trend:** Horizontal bar chart to observe daily sales variations.
- **Quantity by Product:** Pie chart representing product quantities.
- **Interactive Region Slicer:** Filter charts by region.
- **Total Quantity by Product:** Column chart summarizing total quantities.


Created with Microsoft Excel (Pivot Tables, Charts, and Slicers) for sales analysis and visualization.
